Lihat Fitur

Setup

Mulai cepat dengan panduan langkah demi langkah untuk mengatur situs Anda di GitHub Pages.

  1. Buat repository baru di GitHub.
  2. Tambahkan file index.html, styles.css, dan script.js.
  3. Aktifkan GitHub Pages di Settings > Pages dengan branch main.
  4. Gunakan URL hosting yang disediakan untuk melihat situs Anda online.

Biaya

GitHub Pages gratis untuk situs pribadi dan proyek, sehingga kamu bisa mempublikasikan tanpa biaya tambahan.

  • Hosting statis: gratis
  • Domain GitHub Pages: gratis
  • Custom domain: bisa digunakan tanpa biaya GitHub tambahan

Bandwidth

GitHub Pages cocok untuk website sederhana dengan traffic menengah.

  • Bandwidth harian cukup untuk blog, portofolio, dan landing page.
  • Untuk penggunaan tinggi, gunakan CDN atau layanan hosting tambahan.
  • Pastikan ukuran file kecil untuk performa optimal.

Backend

Karena GitHub Pages hanya mendukung konten statis, backend bisa diintegrasikan dengan layanan eksternal.

  • Gunakan API pihak ketiga untuk formulir, komentar, atau autentikasi.
  • Atau tambahkan serverless function melalui Netlify, Vercel, atau Cloudflare.
  • Contoh: kirim data formulir ke layanan form processing tanpa server sendiri.
